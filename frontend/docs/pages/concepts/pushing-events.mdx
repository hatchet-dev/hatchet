# Pushing Events

You can push an event to Hatchet by calling the `push` method on the Hatchet event client and providing the event name and payload. Any tasks that have registered an [event trigger](/concepts/run-on-event) for that event key will be run.

<Callout type="info">
  Event triggers evaluate tasks to run at the time of the event. If an event is
  received before the task is registered, the task will not be run.
</Callout>
