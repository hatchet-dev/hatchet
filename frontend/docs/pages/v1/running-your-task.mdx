# Running Tasks

You can invoke tasks and workflows in a number of ways.

## Run and wait

Invoke a task and block until it completes and returns a result. Use this for synchronous workflows like fanout, LLM calls, or any time the task's output is needed before continuing.

<UniversalTabs items={["Python", "Typescript", "Go", "Ruby"]}>
  <Tabs.Tab title="Python">

Call `run` or `aio_run` on a `Task` or `Workflow` object to invoke it. These methods block until the task or workflow completes and return the result.

<Snippet src={snippets.python.child.trigger.running_a_task} />

The run methods in Hatchet's Python SDK also have async flavors you can `await`. These are prefixed with `aio_`, such as `aio_run`.

<Snippet src={snippets.python.child.trigger.running_a_task_aio} />

Note that the type of `input` here is a Pydantic model that matches the input schema of the task or workflow being triggered.

  </Tabs.Tab>
  <Tabs.Tab title="Typescript">

Call `run` on the `Task` object to invoke it. This returns a promise that resolves when the task completes and returns the result.

<Snippet src={snippets.typescript.simple.run.running_a_task} />

  </Tabs.Tab>
  <Tabs.Tab title="Go">

Call `Run` on the `Task` object to invoke it. This blocks until the task completes and returns the result.

<Snippet src={snippets.go.simple.main.running_a_task} />

  </Tabs.Tab>
  <Tabs.Tab title="Ruby">

Call `run` on the `Task` object to invoke it. This blocks until the task completes and returns the result.

    <Snippet src={snippets.ruby.child.trigger.running_a_task} />

  </Tabs.Tab>
</UniversalTabs>

  <Tabs.Tab title="Ruby">

<Callout type="info">
  While you can run multiple tasks in parallel using the `Run` method, this is
  not recommended for large numbers of tasks. Instead, use [bulk run
  methods](/v1/bulk-run) for large parallel task execution.
</Callout>

## Fire and forget

Fire-and-forget-style triggering enqueues a task without waiting for the result. This is useful for background jobs like sending emails, processing uploads, or kicking off long-running pipelines.

<Snippet src={snippets.python.trigger_methods.workflow.define_a_task} />
