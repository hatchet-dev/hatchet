# Region availability

Hatchet Cloud is available in multiple regions so you can run workloads close to your users and data.

## Current regions

**Hatchet Cloud** ([cloud.hatchet.run](https://cloud.hatchet.run)) Control Plane is deployed in **aws-us-west-2** (Oregon).

We are expanding Data Plane availability. Planned or available regions include:

| Region             | Location         | Status       |
| ------------------ | ---------------- | ------------ |
| aws-us-west-2      | Oregon (US)      | **Live**     |
| aws-us-east-1      | N. Virginia (US) | **Live**     |
| aws-eu-west-1      | Ireland          | **Live**     |
| aws-ap-southeast-2 | Sydney           | Private Beta |

## Request a region

We are always open to rolling out new regions based on demand. If you need a specific region for latency or compliance, [contact us](https://hatchet.run/contact) and we can discuss availability.
