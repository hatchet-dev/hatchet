# Runnables

Runnables are the core building blocks in Hatchet — tasks you define and the various ways to trigger them.

- [Tasks](/v1/runnables/tasks) — Define units of work
- [Run and Wait Trigger](/v1/runnables/run-with-results) — Trigger and wait for results
- [Run Without Wait Trigger](/v1/runnables/run-no-wait) — Fire-and-forget execution
- [Scheduled Trigger](/v1/runnables/scheduled-runs) — Run at a specific time
- [Cron Trigger](/v1/runnables/cron-runs) — Run on a recurring schedule
- [Bulk Run Many](/v1/runnables/bulk-run) — Trigger many runs at once
- [Webhooks](/v1/runnables/webhooks) — Trigger via HTTP webhooks
- [Inter-Service Triggering](/v1/runnables/inter-service-triggering) — Trigger across services
