# Triggers

Additional trigger patterns for running tasks on schedules, in bulk, via webhooks, or across services.

- [Scheduled Runs](/v1/scheduled-runs) — Run at a specific time
- [Cron Runs](/v1/cron-runs) — Run on a recurring schedule
- [Bulk Runs](/v1/bulk-run) — Trigger many runs at once
- [Webhooks](/v1/webhooks) — Trigger via HTTP webhooks
- [Inter-Service Triggering](/v1/inter-service-triggering) — Trigger across services
