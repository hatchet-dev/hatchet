# Use Cases

<Callout type="info">
  These guides assume you have the Hatchet SDK installed and a worker running.
  If you haven't done that yet, start with [Get Started](/get-started).
</Callout>

Common use cases for Hatchet, organized into three categories:

- **AI & Agents**: AI agents, human-in-the-loop approvals, RAG pipelines, and LLM chains
- **Data & Processing**: Batch processing, document extraction, and scheduled jobs
- **Real-Time & Streaming**: Webhook processing, event-driven systems, and live streaming
